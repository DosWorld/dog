Installing DOG

Step one

Download the DOGX.ZIP file.

Step two

Unzip the DOGX.ZIP file. Type these commands EXECTLY as shown.
md DOG
copy dogx.zip \DOG
del dogx.zip
cd dog
unzip DOGX.ZIP
You can use another directory if you wish. Make sure your UNZIP.EXE is in your path. UNZIP.EXE is a program you can use to uncompress the DOGX.ZIP archive.

Step three

Add the following line in your CONFIG.SYS-file. If you already have a shell command in CONFIG.SYS, just replace it with the following. Yuo can make it a comment by adding REM infront of it (in case you want to go back to the old shell. The environment given is just an example. Y can use any value you normally use for COMMAND.COM
SHELL=C:\DOG\DOG.COM -E:512 -P

Step four

Reboot your computer by pressing CONTROL-ALT-DELETE
