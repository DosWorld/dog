DOG DEVELOPER

Welcome to the developers' pages!

Dog is a replacement for COMMAND.COM that comes with DOS 3.3 This section is devoted to the development of DOG.

DOG is made in C and assembler with an i8086 as the target platform. It runs currently on any DOS version from 3.3 and up excluding 4.?
DOG still has many BUGs in it so it will still need development.
